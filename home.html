Inicio

Las etiquetas HTML5 le dieron un nuevo sentido a la web, se flexibilizó y se hizo más dinámica, añadiendo nuevos controles y formas de organizar y mostrar contenido. En esta OVA le servirá de base para empezar.

Objetivo General

Conocer las principales etiquetas HTML 5 para organizar de manera semantica el contenido de una página web.

Objetivos específicos

  • Manejar los contenidos generales con la etiqueta SECTION.
  • Ubicar correctamente el contenido para la cabecera y el pie de página, etiquetas HEADER y FOOTER respectivamente.
  • Establecer el menú dentro de la etiqueta ASIDE.
  • Identificar el contenido principal para ubicarlo dentro de la etiqueta ARTICLE.
