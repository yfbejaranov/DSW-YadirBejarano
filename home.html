Inicio

Vamos a ver dos formas adicionales de aplicar estilos CSS: en la parte inicial del documento HTML (aplicación de estilos interna o CSS interno) o en un archivo de extensión .css independiente del archivo HTML (aplicación de estilos externa o CSS externo).

Objetivo General

El objetivo principal de este sitio es dar a conocer qué es y para qué sirve el lenguaje CSS en el marco del desarrollo WEB.

Objetivos específicos

  • Dar a conocer las características y ventajas que ofrece el uso de hojas de estilos.
  • Dar a conocer el uso de estilos internos.
  • Dar a conocer el uso de estilos externos.
  • Dar a conocer el uso de Selectores CSS.
